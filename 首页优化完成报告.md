### 2. 组件层优化 - ProductCard.tsx

#### 全新卡片设计
- **图片展示区**：
  - 高度：手机 160px，桌面 192px
  - 支持产品图片或渐变背景+emoji
  - 图片悬停时放大效果 (scale-105)
  - 优化的Next.js Image组件，自动webp/avif转换

- **标签系统**：
  - 图片上方显示：价格类型标签、试用标签
  - 半透明白色背景 (backdrop-blur-sm)
  - 色彩区分：会员专属(橙色)、单独购买(蓝色)、可试用(绿色)

- **内容区域**：
  - 标题：悬停变橙色，单行截断
  - 描述：两行截断，避免高度不一致
  - 会员等级徽章：带icon和颜色编码
  - 价格标签：月费/年费/买断分类显示

- **功能特性**：
  - 桌面端：显示前3项功能，带勾选图标
  - 移动端：简化显示"X项功能特性"，节省空间
  - 使用icon增加视觉吸引力

- **CTA按钮**：
  - 灰色底色，悬停变橙色
  - 圆角按钮，shadow效果
  - 响应式字体大小

#### 响应式优化
- 移动端优先设计
- 卡片高度自适应，避免错位
- 图片比例优化，适配不同屏幕
- 触摸友好的按钮大小

### 3. 页面层优化 - page.tsx

#### 删除重复展示
- 删除了原有的"单独购买"独立区域（133-179行）
- 统一使用ProductCard组件展示所有产品

#### 网格布局优化
- 从 `xl:grid-cols-4` 改为 `lg:grid-cols-3`
- 移动端：1列
- 平板：2列
- 桌面：3列
- 卡片间距：手机 16px，桌面 32px

### 4. 配置优化 - next.config.js

#### 图片域名白名单
```javascript
images: {
  remotePatterns: [
    {
      protocol: 'https',
      hostname: 'images.unsplash.com',
      pathname: '/**',
    },
  ],
  formats: ['image/avif', 'image/webp'],
}
```

支持Unsplash图片加载，自动转换为现代图片格式。
