# 宇硕短线 SaaS 平台 - 使用说明

## 项目概述

这是一个完整的单文件 HTML SaaS 平台，包含会员系统、激活码系统、权限控制和后台管理。

**技术栈**: React 18 + Tailwind CSS + LocalStorage
**文件**: 仅 `index.html` 一个文件（约 1700+ 行代码）

## 使用流程

### 普通用户流程
1. **注册账号** → 点击导航栏"登录" → 切换到"注册" → 填写信息
2. **登录** → 使用注册的用户名和密码登录
3. **激活会员** → 进入"个人中心" → 输入激活码（如 `YS-Q-DEMO`）
4. **使用产品** → 返回首页 → 点击产品卡片 → 点击"立即使用"

### 管理员流程
1. **登录管理员账号** → 用户名 `admin` / 密码 `admin888`
2. **进入管理后台** → 导航栏点击"管理后台"
3. **生成激活码** → 选择类型和数量 → 点击"生成并下载"
4. **管理用户** → 切换到"会员管理" → 冻结/解冻用户 → 延长会员时长
5. **查看统计** → 切换到"数据统计" → 查看用户和激活码数据

## 权限说明

### 产品访问权限矩阵

| 产品 | 游客 | 月费 | 季度 | 年费 | 买断 |
|------|------|------|------|------|------|
| 学习圈 | ❌ | ✅ | ✅ | ✅ | ❌ |
| 情绪表格 | ❌ | ❌ | ❌ | ❌ | ✅ |
| 板块助手 | ❌ | ✅ | ✅ | ✅ | ❌ |
| 板块节奏系统 | ❌ | ❌ | ✅ | ✅ | ❌ |
| 复盘系统 | ❌ | ❌ | ❌ | ✅ | ❌ |
| 心理测评系统 | ❌ | ❌ | ❌ | ✅ | ❌ |

### 特殊规则
- **买断权益**：使用 `YS-T-XXXX` 激活后，永久拥有情绪表格访问权，即使会员过期也保留
- **权限叠加**：升级会员时，保留已有的买断权益
- **会员过期**：会员过期后自动降级为"游客"，但买断权益不受影响

## 技术细节

### 数据结构

#### 用户数据（LocalStorage: `ys_users`）
```javascript
{
  id: 1,                          // 用户ID
  username: "test",               // 用户名
  password: "123456",             // 密码（演示项目，实际应加密）
  email: "test@example.com",      // 邮箱
  role: "member",                 // 角色: guest/member/admin
  tier: "quarterly",              // 等级: guest/monthly/quarterly/yearly
  status: "active",               // 状态: active/paused
  expiry: "2026-04-06T...",       // 到期时间（ISO格式）
  specialPerms: ["emotion-table"],// 买断权益数组
  createdAt: "2026-01-06T...",    // 注册时间
  lastLogin: "2026-01-06T..."     // 最后登录时间
}
```

#### 激活码数据（LocalStorage: `ys_activation_codes`）
```javascript
{
  code: "YS-Q-AB12",              // 激活码
  type: "quarterly",              // 类型
  duration: 90,                   // 有效期（天）
  products: [...],                // 包含的产品ID列表
  isUsed: false,                  // 是否已使用
  usedBy: null,                   // 使用者ID
  usedAt: null,                   // 使用时间
  createdAt: "2026-01-06T...",    // 生成时间
  batchId: "BATCH-1234567890"     // 批次ID
}
```

## 常见问题

### Q1: 激活码不生效？
**A**: 检查格式是否正确（`YS-M/Q/Y/T-XXXX`），确保激活码未被使用。

### Q2: 会员过期后还能访问吗？
**A**: 会员过期后降级为游客，无法访问会员产品。但买断权益（如情绪表格）不受影响。

### Q3: 如何重置数据？
**A**: 打开浏览器开发者工具 → Application → Local Storage → 删除 `ys_*` 开头的所有键 → 刷新页面。

### Q4: 账号被冻结怎么办？
**A**: 联系管理员解冻账号（后台管理 → 会员管理 → 点击"解冻"）。
