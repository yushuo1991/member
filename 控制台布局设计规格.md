## 响应式网格系统

### 桌面端 (1920×1080)
- **顶部统计**: `grid-cols-6` - 6列等宽
- **主内容**: `grid-cols-3` - 3列等宽
- **快捷操作**: `grid-cols-4` - 4列等宽
- **总高度**: ~560px (适配1080p一屏显示)

### 平板端 (768-1024px)
- 顶部统计: `md:grid-cols-3` - 降为3列
- 主内容: `md:grid-cols-2` - 降为2列
- 快捷操作: `md:grid-cols-2` - 降为2列

### 移动端 (<768px)
- 所有网格: `grid-cols-1` - 单列堆叠

## 间距规格

### 整体布局
- 容器padding: `p-6` (24px)
- 元素间距: `space-y-4` (16px)
- 卡片间距: `gap-3` (12px)

### 统计卡片
- 内边距: `p-4` (16px)
- 圆角: `rounded-xl` (12px)
- 图标尺寸: `w-5 h-5` (20×20px)
- 数字大小: `text-3xl` (30px)

### 内容卡片
- 内边距: `p-4` (16px)
- 圆角: `rounded-xl` (12px)
- 列表项间距: `space-y-1.5` (6px)
- 头像尺寸: `w-7 h-7` (28×28px)

### 快捷操作
- 内边距: `p-4` (16px)
- 圆角: `rounded-xl` (12px)
- 图标容器: `w-10 h-10` (40×40px)

## 文字规格

### 标题层级
```
页面标题: text-2xl font-bold (24px)
区块标题: text-sm font-semibold (14px)
用户名称: text-xs font-medium (12px)
辅助信息: text-xs text-gray-500 (12px)
```

### 数据显示
```
大数据: text-3xl font-bold (30px) - 统计卡片
中数据: text-2xl font-bold (24px) - 收入/激活码
小数据: text-sm font-semibold (14px) - 列表数据
```

## 交互效果

### 悬停状态
```css
统计卡片: hover:shadow-lg transition-shadow
快捷操作:
  - border: hover:border-[#007AFF]
  - background: hover:shadow-md
  - 图标背景: group-hover:bg-[#007AFF]
  - 图标颜色: group-hover:text-white
列表项: hover:bg-gray-50
```

### 过渡动画
```css
全局: transition-all duration-200
快捷操作: transition-colors duration-300
```

## 可访问性

- 所有图标使用SVG,支持屏幕阅读器
- 使用语义化HTML标签
- 链接和按钮有清晰的hover状态
- 颜色对比度符合WCAG 2.1 AA标准

## 性能优化

- 并行API请求: Promise.all([stats, members, codes])
- 响应式图片: 头像使用CSS渐变背景
- 懒加载: 大数据集分页显示
- 客户端缓存: React状态管理

## 浏览器兼容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 移动端适配

- 触摸友好: 按钮最小44×44px
- 响应式字体: 使用rem单位
- 滚动优化: 单列布局,垂直滚动
- 减少动画: 移动端禁用部分动画
